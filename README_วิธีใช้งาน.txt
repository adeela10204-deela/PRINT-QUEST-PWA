วิธีใช้งาน PRINT QUEST PWA

1) แตกไฟล์ ZIP แล้วอัปโหลดทั้งโฟลเดอร์ขึ้น GitHub Pages, Google Firebase Hosting, Netlify, Vercel หรือเว็บโฮสต์ที่รองรับ HTTPS
2) เปิดลิงก์ด้วย Chrome / Edge / Safari บนมือถือ แท็บเล็ต iPad คอมพิวเตอร์ หรือจอทีวี
3) กด Add to Home Screen / Install App เพื่อติดตั้งเป็นแอป PWA
4) หลังเปิดครั้งแรก เกมจะถูกบันทึกไว้ในเครื่อง สามารถเปิดซ้ำได้แม้ไม่มีอินเทอร์เน็ตบางกรณี

หมายเหตุสำคัญ:
- PWA ต้องเปิดผ่าน HTTPS หรือ localhost เพื่อให้ Service Worker ทำงานเต็มรูปแบบ
- หากเปิดไฟล์ index.html โดยตรง เกมยังเล่นได้ แต่ฟังก์ชันติดตั้งเป็นแอปและออฟไลน์แคชอาจไม่ทำงาน เพราะเป็นข้อจำกัดของเบราว์เซอร์
